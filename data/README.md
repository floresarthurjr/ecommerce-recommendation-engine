# data/

The source dataset (Customer Shopping Trends) is downloaded at runtime
via kagglehub in the notebook. It is not stored in this repository.

When the notebook runs, it produces:
- train_set.csv (3,120 rows, 80% stratified split)
- test_set.csv (780 rows, 20% stratified split)
- full_dataset_with_age_group.csv (3,900 rows with derived age_group column)

Source: https://www.kaggle.com/datasets/iamsouravbanerjee/customer-shopping-trends-dataset
